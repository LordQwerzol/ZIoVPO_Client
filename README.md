# ZloVPO Client

**Учебный проект по дисциплине "Защита информации от вредоносного программного обеспечения"**

Клиент для проекта [сервера](https://github.com/LordQwerzol/ZIoVPO_Server).

---
## Технологии и требования

- Среда разработки: **Qt Creator/Visual Studio Code**
- Операционная система: **Windows**
- Язык программирования: **C/C++**
- Фреймворк: **Qt6**
- Система сборки: **CMake**
---

##  **Функциональность**

### Задание 1

1. При запуске добавлять иконку в трей.
2. Левая кнопка по иконке → показать главное окно.
3. Правая кнопка → контекстное меню:  
   - «Открыть» → показать главное окно.  
   - «Выход» → завершить приложение.
4. При пересоздании панели задач иконка восстанавливается.
5. Поддерживать запуск без показа главного окна.
6. Закрытие главного окна → приложение уходит в трей (не завершается).
7. В главном окне: меню «Файл» → «Выход» → завершение работы.
8. Только один экземпляр приложения для одного пользователя (именованный мьютекс). При повторном запуске — выход до добавления иконки.
9. Сборка через **MSBuild** или **CMake**. Артефакт: один `.exe` со всеми зависимостями.
